# 「 Mio　BBS 」开发手册
本论坛主要参考米游社，作为后端开发学习的练手项目。
## 所用工具
- 语言： Go
- web框架：GIN
- Database: MySQL
- ORM： GORM
- 工具：Swagger，
- 可能选用的工具: kafka，Redis，ORM，

## 项目目标
### 用户管理模块
- 多设备同时登陆
- 自动续期（token？
- 注册
- 用户管理（可选
### 内容管理模块
- 发布帖子
- 帖子显示
- 帖子搜索（可选
